## ✨ Core Highlights

#### 🗂️ Three Skill Categories

All Skills are organized into three clear categories so you can find exactly what you need:

| Category | Description | Examples |
|----------|-------------|----------|
| 💼 **Office Productivity** | Universal skills every knowledge worker needs | PPT design, Excel analysis, writing polish, meeting management |
| 🎯 **Specialist Skills** | Deep domain-specific workflows | Team management, HR tools, design, WeChat articles, Xiaohongshu content |
| 🔧 **Essential Tooling** | Foundation capabilities for your AI workspace | Self-evolution, token management, Skill creation, security |

#### ⚡ The FAST Principle

A built-in decision framework to determine what work can be "Skill-ified":

- **F**ixed — Can the rules be defined?
- **A**utomatable — Can the process run without human judgment?
- **S**tandardized — Can the task be replicated at scale?
- **T**estable — Can the deliverable be verified?

If a task passes all four tests, it's a perfect candidate for an AI Skill.

#### 🚀 One-Click Access

Every Skill card links directly to its GitHub page or Landing Page — click to preview, read the docs, and install instantly. No hunting, no guessing.

## 🛠️ Quick Start

**Step 1: Install dependencies**

```bash
npm install
```

**Step 2: Start dev server**

```bash
npm run dev
```

**Step 3: Build for production**

```bash
npm run build
```

> 💡 Or simply visit the [Landing Page](https://muippt.github.io/mu-skill-hub/) — no installation needed to browse Skills.
