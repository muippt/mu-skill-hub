## ⚡ The FAST Principle

A built-in decision framework to determine what work can be "Skill-ified":

- **F**ixed — Can the rules be defined?
- **A**utomatable — Can the process run without human judgment?
- **S**tandardized — Can the task be replicated at scale?
- **T**estable — Can the deliverable be verified?

If a task passes all four tests, it's a perfect candidate for an AI Skill.

## 🛠️ Quick Start

**Step 1: Install dependencies**

```bash
npm install
```

**Step 2: Start dev server**

```bash
npm run dev
```

**Step 3: Build for production**

```bash
npm run build
```

> 💡 Or simply visit the [Landing Page](https://muippt.github.io/mu-skill-hub/) — no installation needed to browse Skills.
